# OpenCLI Plugins

Monorepo collection of OpenCLI plugins.

## Available Plugins

| Plugin Name | Description |
|-------------|-------------|
| [ctrip-flights](./packages/ctrip-flights/) | Query domestic flight information from Ctrip (携程) |

## Installation

Install individual plugins:

```bash
opencli plugin install github:terryma2024/opencli-plugins/packages/<plugin-name>
```

Example:
```bash
opencli plugin install github:terryma2024/opencli-plugins/packages/ctrip-flights
```
